# Verix Demo-Day Plan

## Vertical Demo (recommended)

**Data-fetching agents for trading desks**

- Buyer AI requests live market data (e.g. LINK/USD).
- Worker returns signed output.
- Verix verifies schema + identity + condition.
- XRPL escrow settles only on valid output.
- Reputation anchor updates DID-linked trust score.

## Integrate in 5 minutes flow

1. Install `@verix/sdk`.
2. Run one live task via `streamDemo()`.
3. Open proof links (EscrowCreate, EscrowFinish, Audit, Reputation Anchor).
4. Run bad-worker mode once to show funds protection.

"Verix is Stripe-for-agent-settlement: DID identity, trustless escrow payout, and on-chain anchored reputation."
